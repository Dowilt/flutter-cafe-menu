# Кафе "У Flutter" — Приложение меню кафе

Лабораторная работа №3. Flutter-приложение для просмотра меню кафе с корзиной заказов.

## Описание

Веб-приложение "Меню кафе" на Flutter с бэкендом на FastAPI и базой данных PostgreSQL.

### Функциональность

- **Каталог меню** — 5 категорий блюд (Завтраки, Супы, Основные блюда, Напитки, Десерты)
- **Детальные карточки** — описание, состав, выбор количества
- **Корзина заказов** — добавление, удаление, очистка с сохранением в БД
- **Поиск по меню** — фильтрация блюд по названию в реальном времени
- **Персистентность данных** — корзина сохраняется в PostgreSQL и переживает перезапуск контейнеров

### Технологии

| Компонент | Технология |
|-----------|-----------|
| Фронтенд | Flutter Web |
| Бэкенд | Python FastAPI |
| База данных | PostgreSQL 15 |
| Веб-сервер | Nginx |
| Контейнеризация | Docker Compose |

## Запуск

### Требования

- Docker и Docker Compose

### Команда запуска

```bash
docker-compose up -d --build
```

Приложение будет доступно по адресу: **http://localhost**

### Остановка

```bash
docker-compose down
```

> Данные корзины сохраняются в Docker volume `pgdata` и не теряются при перезапуске.

## API эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/cart` | Получить содержимое корзины |
| POST | `/api/cart` | Добавить блюдо в корзину |
| DELETE | `/api/cart` | Очистить корзину |
| DELETE | `/api/cart/{id}` | Удалить элемент из корзины |
| GET | `/api/health` | Проверка состояния сервера |
